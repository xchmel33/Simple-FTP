Installation (linux only):
make client
make server

Usage:
./ipk-simpleftp-client {-i ip} {-p port} [-f working_dir] [-u database]
./ipk-simpleftp-server [-h ip] {-p port} [-f working_dir]

RFC 913 commands:
DONE, USER, ACT, PASS, TYPE, LIST, CDIR, KILL, NAME, RETR
